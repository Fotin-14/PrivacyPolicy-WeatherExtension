Privacy Policy

This Chrome extension, "mini Weather Widget", does not collect, store, or share any personal or sensitive user data.

All data used by the extension (e.g., selected city, widget position, widget state, and last-fetched weather info) is stored locally in the user's browser using chrome.storage.local.

No data is transmitted to any third parties for tracking or profiling purposes.

The extension may send requests to an external weather API server for the sole purpose of retrieving weather data, but it does not include any personally identifiable information.
